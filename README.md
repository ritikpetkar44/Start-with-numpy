## 📂 Repository Structure

Yeh repo step-by-step NumPy concepts cover karta hai:

* `01_importation_&_Creation.py` → NumPy import aur array creation
* `02_datatype.py` → Array ka datatype check karna
* `03_shape_&_size.py` → Shape aur size samajhna
* `04_arrangement_of_mat.py` → Array elements modify karna
* `05_zeros_&_ones_matrix.py` → Zero aur ones matrix banana
* `06_reshape_&_identity_matrix.py` → Reshape + identity matrix
* `07_arange_&_linspace_1dMAT.py` → Equal spacing arrays
* `08_empty_&_ravel.py` → Empty matrix aur flatten
* `09_transpose_&_count_arr.py` → Transpose + non-zero count
* `10_argmax_argmin_argsort.py` → Max, Min, Sorting
* `11_add_sub_mul_div.py` → Basic arithmetic operations
* `12_sq_&_sqrt.py` → Square aur square root
* `13_axix_sum.py` → Row-wise & column-wise sum
